Crystal Collector Game

Overview:
    In this game, a number is randomly generated using a Math.floor(Math.random()) function, which becomes the number that the player has to reach to win. To reach the randomly selected number, 4 pictures are presented and given values. Each value is different and added together to reach the random number. If the random number is passed by the sum of the picture values that user will lose the game. In a loss, a new random number is selected, a loss is added to the counter and the game resets.
